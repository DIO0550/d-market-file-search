# File Search Skill

## 概要

ファイル検索を専門エージェントに委譲し、効率的で構造化された検索結果を提供するスキル。

## 自動適用

このスキルは **PreToolUseフック** により、Glob/Grepツール使用時に自動的にトリガーされます。

## 実行方法

### Task tool での呼び出し

```
Task tool:
  subagent_type: "Explore"
  description: "ファイル検索"
  prompt: |
    ファイル検索エージェントとして検索を実行してください。

    検索対象: {検索したい内容}

    plugins/file-search-plugin/agents/file-search-agent.md の指示に従い、
    結果を構造化して報告してください。
```

### 検索リクエストの例

- 「UserService クラスを探して」
- 「認証関連のファイルを検索」
- 「エラーハンドリングのパターンを特定」
- 「API エンドポイントの一覧を取得」
- 「設定ファイルを全て列挙」

## 参照エージェント

詳細な検索手順は `agents/file-search-agent.md` を参照。

## 期待される結果

- 検索マッチ数のサマリー
- 関連度順のファイル一覧
- 重要なコードスニペット
- プロジェクト構造の情報
- 言語固有の最適化適用

## メリット

1. **除外パターン自動適用**: node_modules, target等を自動除外
2. **関連度分析**: 単なるマッチではなく意味のある結果
3. **構造化出力**: 整理されたレポート形式
4. **言語最適化**: TypeScript/Rust等に応じた検索パターン
